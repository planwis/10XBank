# Readme
**For 10XBank contract**
1. install foundry
2. run `yarn`
3. run `yarn test:foundry` for unit testing

**For 10XBank interface**
1. run `yarn`
2. run `yarn start`
3. open `localhost:3000` or the one specified in the commandline in browser
